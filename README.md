# 🍅 Pomodoro

## Instruções

🚀 Para rodar a aplicação é necessário acessar a pasta seja ela `frontend` ou `backend`:

1. Instale as dependências do projeto:

```sh
npm i
```

2. Rode a aplicação:

```
npm run dev
```

2.1 Ambientes:

```
frontend: localhost:4000
backend: localhost:3333
```
